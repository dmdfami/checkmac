# CheckMac

Kiểm tra chất lượng MacBook cũ — báo cáo tiếng Việt.

## Chạy

```bash
curl -sL dmdfami.com/checkmac | bash
```

## Kiểm tra gì?

- Pin (chu kỳ sạc, sức khỏe, nhiệt độ)
- Ổ cứng (SMART, tốc độ đọc/ghi)
- Bảo mật (MDM, Find My Mac, SIP, FileVault, Firmware Password)
- CPU stress test (tất cả nhân, nhiệt độ trước/sau)
- Màn hình, Camera, Loa, Micro, Bàn phím, Trackpad
- WiFi, Bluetooth, USB, Thunderbolt
- Crash logs, Kernel panics
- Điểm chất lượng tổng /10
